¿Te gustaría saber programar en python un bot de Telegram con alguna función que te interese?

En este taller vamos a explicar cómo programar en Python a nivel básico (variables, bucles, condicionales, funciones, uso de librerías...), para después lanzarnos a crear un bot de Telegram de ejemplo que nos mandará un mensaje diario a la hora que programemos con información meteorológica. Esto es solo un ejemplo ya que con los conocimientos adquiridos y algún que otro tutorial, serás capaz de hacerte tus propios bots de Telegram.

Requisitos:

  • Tener ganas de aprender
  • Una cuenta de Telegram
  • Portátil con:
    • Python versión 3. Para instalarlo sigue esta guía en https://wiki.python.org/moin/BeginnersGuide/Download
    • Visual Studio Code. Puedes descargarlo en https://code.visualstudio.com/download

Se recomienda traer todos los programas ya instalados previamente para aprovechar mejor el tiempo. Quienes tengáis dificultades con la instalación mandadnos con antelación un mensaje para poderos echar una mano.

El coste del taller será de 5€ para los no soci@s (destinado a pagar los gastos del local) y gratuito para l@s soci@s de Hackerspace Valencia.
