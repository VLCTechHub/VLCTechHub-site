CLI apps are a great way to bundle up utilities and can serve as a great client for a web service (instead of forcing customers to use cURL). Unfortunately, many of them get stuck in the 70s when it comes to usability and end up annoying users instead of making their life easier. This talk will cover some best practices as well as tools/libraries which can make development of CLIs much more comfortable.

The talk will be in English.

About the speaker:

Wojtek is a Senior Software Engineer at Particle, where he was leading the development of dev tools and also builds CLIs of his own. Contributes to Open Source as much as possible, recently enamoured with Kubernetes and gitops.
