AWS enthusiast, this is going to be BIG!

October 3rd, beginning at 8 P.M. at ADEIT Valencia, we welcome you to join the first ever "Levante AWS Meetup" organized by the Alicante and Valencia AWS User groups!

During 2 hours, we will present you the building blocks of your future AWS infrastructure. Experienced and professional AWS architects will show you what the Cloud is made of, beginning with Julio Faerman, technical evangelist at Amazon Web Services, who will present how to make infrastructure as code a real concept using CloudFormation. Next up, Emile Heitor, certified AWS Architect will explain why and how his company moved their datacenter in AWS. And last but not least, the experienced Alicante AWS User group will provide the finest explanation on how to actually bring infrastructure as code to life, this time with Terraform.

Following those presentations we will meet each other around a couple of drinks and tapas so we talk about our projects and perspectives using the AWS Cloud.

Excited yet? Come and join us!

AWS entusiasta, esto va a ser GRANDE!

El día 3 de Octubre, de las 8 a las 10 de la tarde, en ADEIT Valencia, estas bienvenido a juntarte al primero "Levante AWS Meetup" organizado por los grupos de usuarios AWS de Alicante y Valencia!

Durante 2 horas, presentaremos las fundaciones de tu futura infraestructura AWS. Arquitectos AWS con gran experiencia te presentaran con que se hace la Nube; empezaremos con Julio Faerman, evangelista técnico en Amazon Web Services, que nos hablara de "infraestructura como codigo" usando AWS CloudFormation. Despues, Emilio Heitor, arquitecto AWS certificado presentara como y porque su compañía ha migrado su centro de datos en AWS. Finalmente, el grupo de usuarios AWS de Alicante tambien explicara lo que es la "infraestructura como codigo" pero usando Terraform.

A continuación, nos encontraremos con tapas y copas, y podremos hablar de nuestros proyectos y perspectivas con la nube de AWS.

Emocionado ya? Júntate a nosotros!
