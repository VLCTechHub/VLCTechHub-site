¡¡¡¡¡IMPORTANTE!!!! Quitamos el aforo para evitar que haya gente en lista de espera que, debido a fallos de asistencia, podrían haber asistido. Se aceptará entrada hasta llenar la sala.

Bromas aparte, vamos a terminar este año con una gran charla que nos puede ayudar a reducir la dependencia de plugins y mejorar la velocidad de carga de nuestra web en el proceso. ¿Qué tal suena?

Snippets para WordPress

Hernán Fernández nos hablará acerda de cómo utilizar pequeños trozos de código (snippets) para realizar tareas habituales, y de esta manera tratar de disminuir la dependencia de plugins, factor clave a la hora de mejorar el rendimiento de nuestro WordPress.

En la charla hablará sobre:

• Qué son snippets.

• Por qué utilizar snippets y la problemática de tener muchos plugins.

• Algunos recursos donde encontrarlos.

• Cómo utilizar los snippets en WordPress mediante dos métodos: function.php y Code Snippets Plugin (haré hincapié en este último).

• Ejemplos prácticos de uso.

Nota: la charla irá orientada a usuarios finales.

Presentada por Hernán Fernández (https://www.twitter.com/Hernan_Fdez). Hernán es desarrollador web especializado en aplicaciones y WordPress, analista de sistemas informáticos y programador Puedes seguirle en @Hernan_Fdez (https://www.twitter.com/Hernan_Fdez).
