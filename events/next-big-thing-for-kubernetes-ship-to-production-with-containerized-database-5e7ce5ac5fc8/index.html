Kubernetes se ha convertido en el estándar de facto de la industria para la orquestación de contenedores. Kubernetes no solo ha surgido como una herramienta crítica para los equipos de DevOps empresariales, sino también como un marco de trabajo de aplicaciones distribuidas; siendo clave en el aumento de las implementaciones híbridas y multicloud. Si bien la capacidad de ejecutar aplicaciones en la nube utilizando contenedores con Kubernetes ha facilitado el trabajo de todos, esta tecnología en evolución no carece de dificultades, como podría ser la compleja ejecución de las base de datos distribuidas sobre contenedores.

Durante el meetup discutiremos las tendencias clave detrás de la adopción de Kubernetes y mecanismos para manejar base de datos junto a otras aplicaciones de microservicios, todo ello en la misma plataforma.

Tras la charla, tendremos la oportunidad de intercambiar ideas y experiencias.
