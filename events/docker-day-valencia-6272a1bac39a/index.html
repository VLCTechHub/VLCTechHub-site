Este meetup co-organizado con Docker y Valencia DevOps, contará con la presencia de varios empleados de Docker y BigML presentando las sesiones.

Veremos cómo Docker ha cambiado el panorama del software y cómo se utiliza entre la comunidad dev.

AGENDA

  • 18:15 - 18:30: Bienvenida por Docker
  • 18:30 - 19:00: Despliega modelos de Machine Learning en pocos clicks
    • Ponente: Oscar Rovira, Director Web en BigML
    • Descripción: Ya tengo hecho mi modelo predictivo, ahora toca ponerlo en producción. ¿Cómo? ¿Producqué? ¡Pero si yo lo tengo en un iPython Notebook! Tras más de una década ayudando a nuestros clientes a poner sus modelos en producción de forma sencilla y bonita, hemos decidido que era hora de estandarizar este proceso y compartir con el mundo nuestra forma de hacerlo. Por supuesto siguiendo nuestra marca de identidad: hacer el Machine Learning simple, seguro y escalable.
    • Esta charla será en castellano.
  • 19:00 - 19:30: Creating A Landlord for Multi-tenant K8s Using Flux, Gatekeeper, Helm, and Friends
    • Speaker: Michael Irwin, Engineering Manager at Docker
    • Abstract: Supporting multi-tenant environments in Kubernetes is easy, right? Well, it can be. But, it takes organization, structure, and proper policy enforcement. At Virginia Tech, we've been building a "Common Application Platform" that gives each tenant its own manifest repo and deploys those manifests into isolated namespaces using Flux. By leveraging Gatekeeper and Karpenter, we can properly isolate workloads into node pools and ensure tenants don't step on each other's toes. And best of all, our tenant config is in a simple Helm chart that we call "the landlord." In this talk, we'll dive into how we've built the landlord, the various policies and mutations we're using, and how it works... all with the intent that you can build your own platform too! We'll have live demos and even try to break a thing or two!
    • This session will be presented in English.
  • 19:30 - 20:45 Networking, picoteo y swag.

Este evento está patrocinado por Flywire y Tinybird.
