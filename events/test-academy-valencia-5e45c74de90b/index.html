Test Academy celebrará su jornada de master class en Valencia el 04 de marzo de 2020. Tiene como objetivo reunir a los profesionales y expertos para un día de aprendizaje, intercambio de experiencias y conocer nuevos compañeros dentro de la comunidad de testing de software en España.

Para confirmar tu asistencia debes comprar la entrada a través de nuestra web oficial. No se confirmará tu asistencia al evento hasta que no hayas hecho el pago y hayas obtenido tu entrada.

Puedes elegir entre 8 master class con diferentes temáticas:

  • Testing Ágil (Albert Tort)
  • QA + DevOps (Ard Kramer)
  • Gestion y Estrategias de Testing (Toni Robres y Javier Amate)
  • DevOps + Kubernetes (Fernando Llaca)
  • Testing Continuo + Inscpeccion Continua (Oscar Moreno)
  • DevOps + Automatización de Pruebas (Xavier Hidalgo)
  • Automatización de Pruebas (Fran Guerrero)
  • Testing en producción (David Lluna)
