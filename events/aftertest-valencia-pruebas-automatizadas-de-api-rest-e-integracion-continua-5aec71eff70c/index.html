Desde nexo QA empezamos nuestro AfterTest en Valencia, con una charla sobre pruebas automatizadas de API REST e Integración Continua.

Una de las claves de éxito de un buen proceso de pruebas son las herramientas que utilizamos no solo para apoyar el proceso de pruebas sino automatizar las actividades de testing que realizamos día a día.

En esta charla, Rodrigo Jiménez nos hablará sobre la integración contínua y la automatización de testing funcional para un API REST. Sobre todo, cómo montar una cadena de herramientas integradas para lanzar estas pruebas, generar defectos y reportar los resultados automáticamente utilizando Jenkins, JMeter, TestLink y Jira.

Rodrigo, profundizará sobre cómo crear un plan de pruebas con jmeter, para que su posterior ejecución desde Jenkins genere un report que pasearemos a formato “tap”. Estos resultados se plasmarán en una nueva ejecución en TestLink (herramienta elegida como test management) u otra herramienta que soporte API REST. Si el resultado de las pruebas no es positivo, se abrirá un defecto en Jira con el mensaje de error definido en JMeter que será asignada a la última ejecución del test correspondiente en TestLink.

¡Únete a nuestra comunidad! Ven y disfruta una buena charla, una cerveza y hacer networking con otros testers.

Quedamos en el Innovation Center Valencia, a partir de las 18:30h, Calle San Vicente Mártir, 148.

Después de la presentación haremos un concurso entre los participantes y se podrá ganar una entrada a expo:QA'18 (Madrid, 4-6 de Junio 2018).

Este AfterTest está patrocinado por SOGETI
