  • ¿Eres SRE y te gustan las consolas en modo texto? Te gusta Steampipe

  • No soy un SQL guru. ¿Steampipe es para mí?
  • Las consultas más populares para el API de AWS
  • Steampipe Mods: dashboards and benchmarks
  • Steampipe checks. ejemplos de Compliance.

SPEAKER:

Manuel es PITC, pintor impresionista con titulo certificado, y fan de Javi Moreno (@ciberado).

En sus ratos libres, se dedica al noble arte de la gestion del cloud. Todas las mañanas, después de autoescalarse, toca su arpa Kubernetes para coger inspiración.

Disfruta viendo contenedores luchando como gladiadores por los limitados recursos de sus clusters. Esos clusters incomprendidos... Amados y odiados por igual.

Le gusta ese olor a peligro característico de los despliegues con Jenkins.

Le gustan los atardeceres. Escribe poesías mientras monitoriza la nube.
